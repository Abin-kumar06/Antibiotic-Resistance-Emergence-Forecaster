📋 System Components

📊 Data Sources & Generation

  • Patient Profiles: Simulated Demographics including Age, Gender, and Hospital Region context.
  • Clinical Inputs: Core variables spanning specific Medical Infection Types and precisely diagnosed Bacterial Strains (e.g., E. coli, MRSA).
  • Treatment Logs: Actionable treatment features like targeted Antibiotic administration, Milligram Dosage, and multi-day Duration periods mapping to clinical outputs.

⚙️ Processing Pipeline

  • Imputation Engine: Fills missing or null clinical data points using Median (Age) and Mode (Dosage) to mimic real-world Electronic Health Record inconsistencies.
  • Temporal Features: Generates advanced metric rolling averages (e.g., measuring the historical 30-Day Hospital Antimicrobial Usage Rate, and contextual 30-Day Regional Resistance curves).
  • Structure & Matrix Scaling: Converts string categoricals optimally via modular joblib LabelEncoders while standardizing continuous numerical features for algorithmic variance.

🧠 ML Engine & Explainability

  • Predictive Classifiers: Employs diverse baseline and optimized modeling standards (Logistic Regression, Random Forest, lightweight structured end-to-end XGBoost algorithms) to accurately classify binary resistance probabilities.
  • Deep Neural Networks: Adapts scalable tabular intelligence using scikit-learn's MLPClassifier implementations to uncover deeply hidden non-linear intersections across vast medical interactions.
  • SHAP Explainer: Embeds interpretative capabilities via Shapley Additive exPlanations to interpret machine learning outputs globally, enabling clinicians to visually decode model factors in real-time.

🚀 Applications & Deployment Layer

  • API Layer (FastAPI): A high-concurrent backend router serving predictive mathematical outputs sequentially using XGBoost classification thresholds via robust JSON REST pathways.
  • Monitoring Dashboard (Streamlit GUI): Renders visual interface components to practitioners, triggering active medical insights or immediate critical severity warnings driven securely by algorithmic threshold detections online.
